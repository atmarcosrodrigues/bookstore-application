Livraria JS - REST API


Introdução

Livraria JS - REST API é um free online REST service que descreve métodos de acesso para inclusão, atualização, remoção e manipulação dos recursos providos pela aplicação.

Ir para a aplicação

POST inserirLivro

-Response: Text

Verbo usado para inserir um novo livro no sistema. Exemplo de request:

  -X POST -d 'titulo=ANGULARJS 2&desc=AngularJS: Novice to Ninja is your fast track route to mastering AngularJS, the superheroic JavaScript framework. AngularJS provides the fastest, most efficient way to build single page web applications.&autores=Russell, Norvig&preco=219,90&urlCapa=http://ecx.images-amazon.com/images/I/410Yirblc6L._SX387_BO1,204,203,200_.jpg' http://livraria-js.appspot.com/inserirLivro
                         
  Parâmetros:
    -titulo: Titulo do livro a ser cadastrado
    -desc: Descrição do livro a ser cadastrado
    -preco: Preço do livro a ser livro cadastrado
    -urlCapa: Endereço url para a imagem de capa do livro a ser cadastrado

Exemplo de resposta: (Simples menssagem de confirmação)

  ok

PUT updateLivro

-Response: Text

Verbo usado para atualizar informações de um livro já existente no sistema. Exemplo de request:

  -X PUT -d 'id=XOHZCuKDUCZXdAtg1OFFpXv98&titulo=O Silmarillion UPDATE&desc=O Silmarillion, relata acontecimentos de uma época muito anterior ao final da Terceira Era, quando ocorreram os grandes eventos narrados em O Senhor dos Anéis. São lendas derivadas de um passado remoto, ligadas às Silmarils, três gemas perfeitas criadas por Fëanor, o mais talentoso dos elfos. Tolkien trabalhou nesses textos ao longo de toda a sua vida, tornando-os veículo e registro de suas reflexões mais profundas.&autores=J. R. R. Tolkien&preco=34,90&urlCapa=http://3.bp.blogspot.com/-umADLKwLxNA/VfW-9wmpaKI/AAAAAAAABbU/roqu_6SukSQ/s1600/download.jpg' http://livraria-js.appspot.com/updateLivro

  Parâmetros:
    -id: Identificador de algum já livro cadastrado
    -titulo: Novo titulo para o livro cadastrado
    -desc: Nova descrição para o livro cadastrado
    -preco: Novo preço para o livro cadastrado
    -urlCapa: Novo endereço url para a imagem de capa do livro cadastrado

Exemplo de resposta:

  ok

DELETE deletarLivro

-Response: Text

Verbo usado para remover um livro existente no sistema. O request apaga todas as inforções do livro, incluindo comentários associados a ele. Exemplo de request:

  -X DELETE http://livraria-js.appspot.com/deletarLivro?id=va8XSjYW0adpSS6dLwaCxqlfD
                         
  Parâmetros:
    -idlivro: Identificador de algum livro cadastrado

Exemplo de resposta:

  ok

POST postarComentario

-Response: Text

Verbo usado para inserir um novo comentario no sistema. Exemplo de request:

  -X POST -d 'idlivro=va8XSjYW0adpSS6dLwaCxqlfD&usuario=Gloria Pires&conteudo=Não sou capaz de opinar!' http://livraria-js.appspot.com/postarComentario

  Parâmetros:
    -idlivro: Identificador de algum livro cadastrado 
    -usuario: Nome do usuário que vai postar o comentario 
    -conteudo: Conteúdo do comentário

Exemplo de resposta:

  ok

GET getComentarios

-Response: JSON

Retorna como recurso os comentarios correspondente a um usuário específico, ou todos os comentários inseridos, caso não seja informado nenhum idlivro no request. Exemplos de request:

  GET http://livraria-js.appspot.com/getComentarios
  GET http://livraria-js.appspot.com/getComentarios?idlivro=va8XSjYW0adpSS6dLwaCxqlfD

  Parâmetros:
    -idlivro: Identificador de algum livro cadastrado (Opcional)
